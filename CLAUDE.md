Tauri v2 desktop app for recording course videos.

- React Router v7 SPA mode, React 19, Tailwind CSS v4, shadcn/ui (new-york)
- React Compiler enabled via babel plugin
- SQLite via @tauri-apps/plugin-sql
- OBS WebSocket on ws://localhost:4455
- Asset protocol enabled for local video files
